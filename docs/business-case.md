You work as a data analyst at a school that is implementing an electronic gradebook. 

The administration wants to understand: 
- How to organize data on students, classes, teachers, and grades
- How to create a unified analytical model for all data
- How to prepare data for further analysis of academic performance
- How to ensure accurate between tables
- How to include a timeline for analyzing grade trends
- What the average score is by subject and class
- How student performance has changed over the year
- What the ratio of different types of grades is
- Which classes and subjects show the best results
- How performance is distributed across regions

Your task as an analyst: 
- Go through the full analytical cycle of working with data in Power BI: from importing raw CSV files through cleaning in Power Query to building a proper star schema.
- Create a comprehensive analytical dashboard in Power BI that combines various types of visualizations, properly configured filters, and interactions between charts.
